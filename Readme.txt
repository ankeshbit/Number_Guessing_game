# 🎯 Number Guessing Game (Java)

A fun and interactive console-based Number Guessing Game built in Java.  
The program randomly selects a number between 1 and 100, and the user has to guess it.

---

## 🚀 Features
- Random number generation between 1 and 100
- Hints for "too high" or "too low"
- Counts total attempts
- Option to exit anytime by entering 0
- Input validation for invalid entries

---

## 💻 How to Run
1. Clone this repository or download the `.java` file.
2. Open a terminal or command prompt.
3. Compile the file:
   ```bash
   javac Main.java
